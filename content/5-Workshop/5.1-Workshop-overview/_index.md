# Introduction

## Introduction to AWS SOC Platform

AWS SOC Platform is a Security Operations Center (SOC) solution built on AWS, designed to provide security monitoring, log collection, threat detection, and automated response to security events.

The objective of this workshop is to guide you step-by-step through deploying a multi-layer SOC architecture (Defense in Depth) on AWS by combining networking, security, monitoring, and automation services.

## Workshop overview

- Secure network infrastructure using Amazon VPC.
- Log collection and storage using AWS CloudTrail, VPC Flow Logs, and Amazon S3.
- Threat detection using Amazon GuardDuty, AWS Security Hub, and AWS Config.
- Automated security response using Amazon EventBridge, AWS Lambda, AWS Step Functions, and Amazon SNS.
- System monitoring through Amazon CloudWatch Dashboard.

After completing this workshop, you will understand how to build an AWS SOC platform capable of monitoring, detecting, and responding to security events.
