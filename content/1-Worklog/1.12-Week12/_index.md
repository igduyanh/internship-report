### Week 12 Achievements:

- Successfully developed an Amazon CloudWatch Dashboard to provide centralized monitoring for the entire system.
- Configured the dashboard to visualize key security and operational metrics, including:
  - GuardDuty Findings
  - Security Hub Findings
  - Lambda Invocations
  - Step Functions Executions
  - EC2 Health
  - WAF Blocked Requests
  - CloudFront Requests
  - ALB Request Count
- Successfully configured Amazon CloudWatch Alarms and verified email notifications delivered through Amazon SNS.
- Conducted multiple attack simulation scenarios to assess the system's detection and incident response capabilities.
- Confirmed that the complete Detect → Respond → Recover workflow operated as designed and met the expected functional requirements.
- Optimized IAM Policies according to the Principle of Least Privilege, reducing unnecessary permissions and strengthening system security.
- Evaluated the deployment cost and proposed future enhancements, including Multi-AZ, Multi-Account architecture, Infrastructure as Code (IaC), Amazon Macie, and Amazon Detective.
- Completed the project report, deployment documentation, and user guide, ensuring the system is ready for deployment and future maintenance.
