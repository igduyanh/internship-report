## Mục tiêu

Tập trung các finding bảo mật ở một nơi.

## Giải thích dịch vụ

Security Hub mang lại một dashboard tổng hợp từ nhiều dịch vụ bảo mật.

## Bước 1. Bật Security Hub

- Mở Security Hub Console, click **Go to Security Hub**
- Chọn **Enable Security Hub**.

![Enable Security Hub](/images/5-Workshop/5.5-Detection-Compiance/enable-schub.png)

## Bước 2. Chọn standard

Sau khi enable, chọn **Security standards**:

- Enable **AWS Foundational Security Best Practices v1.0.0**
- Enable **CIS AWS Foundations Benchmark v1.4.0**
- Click **Enable**

## Validation

- Security Hub đã hoạt động.
- Các finding có thể được xem tập trung.

## Chuyển sang bước tiếp theo

Tiếp theo, bạn sẽ tạo các thành phần phản ứng tự động.
