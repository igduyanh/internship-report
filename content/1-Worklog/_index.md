**On this page**, how should you introduce your worklog? How many **weeks** did you take to complete the program? What **did you do** during those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)

**Week 2:** [Learn the core AWS services: IAM, VPC, EC2, and S3 (Lab 1 - 4)](1.2-week2/)

**Week 3:** [Learn and practice Amazon RDS (Lab 5)](1.3-week3/)

**Week 4:** [Learn Load Balancing & Auto Scaling architecture (Lab 6)](1.4-week4/)

**Week 5:** [Cost management and system monitoring with CloudWatch (Lab 7 - 8)](1.5-week5/)

**Week 6:** [Learn and practice infrastructure deployment with Terraform](1.6-week6/)

**Week 7:** [Align on project topic and create implementation plan](1.7-week7/)

**Week 8:** [Establish workflow, project roles, and research technologies](1.8-week8/)

**Week 9:** [Build basic security monitoring environment on AWS (GuardDuty, Security Hub)](1.9-week9/)

**Week 10:** [Build automated incident response system (Lambda, EventBridge, SNS)](1.10-week10/)

**Week 11:** [Finalize automated incident response workflow with AWS Step Functions](1.11-week11/)

**Week 12:** [Build centralized monitoring dashboard and perform attack simulation testing](1.12-week12/)
